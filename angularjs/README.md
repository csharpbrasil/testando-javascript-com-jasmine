# Testando seu código JavaScript com Jasmine
## Exemplo de projeto AngularJS

Para testar o projeto, utilize os seguintes comandos abaixo:

### Instalar os pacotes

> npm install

### Executar os testes unitários

> .\node_modules\.bin\karma start
